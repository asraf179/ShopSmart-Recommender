To create two rows with a 4:1 column ratio (meaning each row has two columns where the first column takes 4 parts and the second takes 1 part), here's how you can do it in different contexts:

✅ 1. Using HTML + Tailwind CSS

html
<div class="grid grid-cols-5 gap-4 mb-4">
  <div class="col-span-4 bg-blue-200 p-4">Row 1 - Column 1 (4/5)</div>
  <div class="col-span-1 bg-green-200 p-4">Row 1 - Column 2 (1/5)</div>
</div>

<div class="grid grid-cols-5 gap-4">
  <div class="col-span-4 bg-blue-200 p-4">Row 2 - Column 1 (4/5)</div>
  <div class="col-span-1 bg-green-200 p-4">Row 2 - Column 2 (1/5)</div>
</div>

✅ 2. Using CSS Flexbox (with custom widths)

html
<div style="display: flex; margin-bottom: 1rem;">
  <div style="flex: 4; background-color: lightblue; padding: 1rem;">Row 1 - Col 1</div>
  <div style="flex: 1; background-color: lightgreen; padding: 1rem;">Row 1 - Col 2</div>
</div>

<div style="display: flex;">
  <div style="flex: 4; background-color: lightblue; padding: 1rem;">Row 2 - Col 1</div>
  <div style="flex: 1; background-color: lightgreen; padding: 1rem;">Row 2 - Col 2</div>
</div>

✅ 3. Bootstrap 5 Example (using 12 columns system)

html
<div class="row mb-3">
  <div class="col-8 bg-primary text-white p-3">Row 1 - Column 1 (4/5)</div>
  <div class="col-4 bg-success text-white p-3">Row 1 - Column 2 (1/5)</div>
</div>

<div class="row">
  <div class="col-8 bg-primary text-white p-3">Row 2 - Column 1 (4/5)</div>
  <div class="col-4 bg-success text-white p-3">Row 2 - Column 2 (1/5)</div>
</div>

(Note: 4:1 ratio → 8:2 or 8:4 out of 12 columns depending on framework)
